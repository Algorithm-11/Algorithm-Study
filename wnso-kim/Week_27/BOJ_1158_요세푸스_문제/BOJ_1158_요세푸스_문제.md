## 풀이
큐를 이용해 순회하며 해결할 수 있다.   
순회 하는 대신 (index+K) % queue.size() 를 이용해 해결했다!

```java
while(list.size() != 1){
    index += K;
    index %= list.size();

    result.append(list.get(index)+", ");

    list.remove(index);
    index--;
}
```
